Copy the *.pot file to a new file with a name corresponding to the language code and the .po extension.
Please note that translation files for some languages have a code consisting only of the language code
(e.g., “uk.po” for Ukrainian), while others have a code consisting of the language code and the country code
separated by an underscore (e.g., “de_DE.po” for German in Germany or “es_PE.po” for Spanish in Peru).

You can find the code you need in your website's database in the wp_options table under the “WPLANG” option value
https://shotify.themerex.net/dC8DkXQ29t

Alternatively, you can open “Settings - General” in your website's admin panel, right-click on the “Site Language” field,
and select “Explore” from the context menu. This will open the developer panel, where you can view the codes for all available languages
